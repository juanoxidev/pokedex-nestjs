# Ejecutar en desarrollo
1. Clonar el repositorio
2. Ejecutar
```bash
$ yarn install
```

3. Tener Nest CLI instalado
```bash
$ yarn install -g @nestjs/cli
```

4. Levantar BD MONGO Docker
```bash
$ docker-compose up -d
```

## Servir contenido estatico

```bash
# crear directorio public 
$ yarn add @nestjs/serve-static
```

## Stack usado
* MongoDB
* Nest
